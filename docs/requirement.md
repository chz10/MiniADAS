# Requirement Summary

MiniADAS-CPP 是一个基于 C++17 的 ADAS 感知目标离线评测工具。

## P0 功能

- CMake 工程；
- BBox / ObjectInfo / FrameData；
- CSV 文件解析；
- IoU 计算；
- 单帧匹配；
- TP / FP / FN 统计；
- `summary.csv` 输出；
- README 文档。

## P1 功能

- `detail.csv` 输出；
- OpenCV 可视化；
- 批量处理；
- 配置文件；
- 日志模块；
- 单元测试。

## P2 功能

- 多线程处理；
- 线程池封装；
- 错误样本截图；
- 匈牙利匹配；
- 性能统计；
- 项目流程图。
